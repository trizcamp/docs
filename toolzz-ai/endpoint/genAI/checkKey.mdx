---
title: 'CheckKey GenAI'
description: Endpoint to create GenAI
api: 'POST https://chat.api.toolzz.com.br/api/v1/genAI/checkKey'
authorization: none
---

<Expandable title="Payload">
<ResponseField name="genAIKey" type="string" required>
  Key for the generative AI configuration.
</ResponseField>
<ResponseField name="genAIOrganization" type="string" required>
  Organization associated with the generative AI.
</ResponseField>
<ResponseField name="genAI" type="string" required>
  Name or type of the generative AI.
</ResponseField>
<ResponseField name="genAIId" type="string">
  Optional unique identifier for the generative AI.
</ResponseField>
</Expandable>
