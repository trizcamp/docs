## Response Description 
Below you will see the description of the get a list score response

<ResponseField name="dataType" type="string" required>
  Type of data (e.g., NUMERIC, BOOLEAN, CATEGORICAL).
</ResponseField>
<ResponseField name="id" type="string" required>
  Unique identifier for the score.
</ResponseField>
<ResponseField name="traceId" type="string" required>
  Trace ID associated with the score.
</ResponseField>
<ResponseField name="name" type="string" required>
  Name of the score.
</ResponseField>
<ResponseField name="source" type="string" required>
  Source of the score (e.g., ANNOTATION, API, EVAL).
</ResponseField>
<ResponseField name="observationId" type="string" required>
  Observation ID associated with the score.
</ResponseField>
<ResponseField name="timestamp" type="date-time" required>
  Timestamp of the score.
</ResponseField>
<ResponseField name="createdAt" type="date-time" required>
  Creation timestamp of the score.
</ResponseField>
<ResponseField name="updatedAt" type="date-time" required>
  Last update timestamp of the score.
</ResponseField>
<ResponseField name="authorUserId" type="string" required>
  User ID of the author who created the score.
</ResponseField>
<ResponseField name="comment" type="string" required>
  Comment associated with the score.
</ResponseField>
<ResponseField name="configId" type="string" required>
  Config ID associated with the score.
</ResponseField>
<ResponseField name="queueId" type="string" required>
  Queue ID associated with the score.
</ResponseField>
<ResponseField name="value" type="integer" required>
  Value of the score.
</ResponseField>
