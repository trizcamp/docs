---
title: 'Get a prompt'
description: Endpoint to get a prompt on toolzz analytics
api: 'GET /api/public/v2/prompts/{promptName}'
authorization: none
---
<Expandable title="Request">
<ResponseField name="promptName" type="string" required>
  The name of the prompt.
</ResponseField>
<ResponseField name="version" type="integer">
  Version of the prompt to be retrieved (nullable).
</ResponseField>
<ResponseField name="label" type="string">
  Label of the prompt to be retrieved (nullable), defaults to "production".
</ResponseField>
</Expandable>

## Response Description 
Below you will see the description of the get prompt response

<ResponseField name="type" type="string" required>
  Type of the interaction (e.g., "chat").
</ResponseField>
<ResponseField name="name" type="string" required>
  Name of the interaction or entity.
</ResponseField>
<ResponseField name="version" type="integer" required>
  Version of the configuration.
</ResponseField>
<ResponseField name="config" type="null">
  Configuration details (nullable).
</ResponseField>
<ResponseField name="labels" type="array" required>
  List of labels associated with the interaction.
  <Expandable title="items">
    <ResponseField name="label" type="string" required>
      A single label.
    </ResponseField>
  </Expandable>
</ResponseField>
<ResponseField name="tags" type="array" required>
  List of tags associated with the interaction.
  <Expandable title="items">
    <ResponseField name="tag" type="string" required>
      A single tag.
    </ResponseField>
  </Expandable>
</ResponseField>
<ResponseField name="prompt" type="array" required>
  List of prompts containing roles and content.
  <Expandable title="items">
    <ResponseField name="role" type="string" required>
      Role of the participant in the interaction (e.g., "user", "assistant").
    </ResponseField>
    <ResponseField name="content" type="string" required>
      Content of the prompt or message.
    </ResponseField>
  </Expandable>
</ResponseField>
