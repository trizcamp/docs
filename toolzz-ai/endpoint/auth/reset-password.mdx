---
title: 'Reset Password'
description: 'Endpoint to change password of institution'
authMethod: "bearer"  
api: 'POST https://chat.api.toolzz.com.br/api/v1/auth/reset-password/:email'
---

<RequestExample>

```json Payload Example
  {
    "password": "<string>",
    "password_confirmation": "<string>"
  }
```
</RequestExample>

## Response Description 
Below you will see the description of the reset password response

<ResponseField name="id" type="uuid" required>
  Unique identifier for the user.
</ResponseField>
<ResponseField name="name" type="string" required>
  First name of the user.
</ResponseField>
<ResponseField name="lastName" type="string" required>
  Last name of the user.
</ResponseField>
<ResponseField name="email" type="string" required>
  Email address of the user.
</ResponseField>
<ResponseField name="token" type="string" required>
  Authentication token for the user.
</ResponseField>
<ResponseField name="institutionId" type="uuid">
  UUID of the associated institution (optional).
</ResponseField>
<ResponseField name="institutionName" type="string">
  Name of the associated institution (optional).
</ResponseField>
<ResponseField name="isActive" type="boolean" required>
  Indicates whether the user's account is active.
</ResponseField>
<ResponseField name="hasPassword" type="boolean" required>
  Indicates whether the user has a password set.
</ResponseField>
<ResponseField name="superAdmin" type="boolean" required>
  Indicates whether the user has super admin privileges.
</ResponseField>
<ResponseField name="verified" type="boolean" required>
  Indicates whether the user's email is verified.
</ResponseField>
<ResponseField name="members" type="array" required>
  List of members associated with the user.
</ResponseField>
<ResponseField name="isMember" type="boolean" required>
  Indicates whether the user is a member of an institution.
</ResponseField>
