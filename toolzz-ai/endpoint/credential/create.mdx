## Response Description 
Below you will see the description of the bot response



<ResponseField name="prompt" type="string" required>
  Text or command used as input for the bot.
</ResponseField>
<ResponseField name="description" type="string" required>
  Detailed description of the bot's purpose or functionality.
</ResponseField>
<ResponseField name="folders" type="any[]" required>
  List of folders associated with the bot.
</ResponseField>
<ResponseField name="plugins" type="any[]" required>
  Active plugins or integrations for the bot.
</ResponseField>
<ResponseField name="botGenAI" type="any" required>
  Settings related to the bot's artificial intelligence.
</ResponseField>
<ResponseField name="assistantId" type="string" required>
  Unique ID of the linked assistant.
</ResponseField>
<ResponseField name="IconFile" type="any" required>
  Icon file associated with the bot.
</ResponseField>
<ResponseField name="widgetId" type="string">
  Optional ID of the linked widget.
</ResponseField>
<ResponseField name="tags" type="string[]">
  List of tags associated with the bot.
</ResponseField>
<ResponseField name="unityId" type="string">
  ID of the organizational unit.
</ResponseField>
<ResponseField name="progressStatus" type="string">
  Progress status of the bot.
</ResponseField>
<ResponseField name="safeMode" type="boolean">
  Indicates whether safe mode is enabled.
</ResponseField>
<ResponseField name="institutionId" type="string" required>
  ID of the institution responsible for the bot.
</ResponseField>
<ResponseField name="indexService" type="string">
  Indexing service linked to the bot.
</ResponseField>
<ResponseField name="isActivated" type="boolean" required>
  Activation status of the bot.
</ResponseField>
