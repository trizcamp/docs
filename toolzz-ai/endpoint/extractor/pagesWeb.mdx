---
title: 'Extract Pages'
description: Endpoint to extract pages from URL
api: 'POST https://api/v2/extractor/web-scraper'
---
<Expandable title="Payload">
<ResponseField name="unityId" type="uuid" required>
  UUID of the associated unity.
</ResponseField>
<ResponseField name="datasetId" type="uuid" required>
  UUID of the dataset associated with the operation.
</ResponseField>
<ResponseField name="url" type="string" required>
  URL for the dataset or resource.
</ResponseField>
<ResponseField name="limit" type="number" required>
  Limit for the number of items or results to retrieve.
</ResponseField>
</Expandable>

<RequestExample>
```json Payload Example
{
  "unityId": "<uuid>",
  "datasetId": "<uuid>",
  "url": "https://www.toolzz.com.br/home#b_386227_1_45971",
  "limit": 5
}
```
</RequestExample>
